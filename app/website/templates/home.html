MexBank

Un banco hecho a tu medida

Esta es una herramienta de gestión para el banco "MexBank" la cual permite gestionar toda su base de datos. Podrás agregar registros, consultarlos y eliminarlos de una manera muy sencilla, con una interfaz intuitiva y fácil de entender.

Para el desarrollo de este proyecto se utlizó el lenguaje de programación python con el framework de desarrollo web Django, se usó MySQL como gestor de bases de datos y Bootstrap para el diseño de la interfaz gráfica.
